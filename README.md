Manual de Usuario: Trivia en Red
1. Administrador (Servidor)
Ejecuta el archivo Servidor.java.

La consola indicará que está esperando jugadores. No hagas nada todavía.

Espera a que todos los clientes se hayan conectado.

Escribe START y pulsa Enter para iniciar la partida.

2. Jugadores (Clientes)
Ejecuta el archivo Cliente.java (se admiten hasta 10 jugadores a la vez).

Escribe tu nombre cuando la consola te lo pida y pulsa Enter.

Espera a que el administrador arranque el juego.

3. Como Jugar
Al empezar, recibirás una pregunta con 4 opciones de respuesta (a, b, c, d).

Tienes exactamente 10 segundos para contestar.

Para responder, es obligatorio escribir la palabra RESPUESTA seguida de un espacio y la letra elegida.

Ejemplo de formato correcto: RESPUESTA a

Al finalizar las 5 rondas, se mostrará el ranking definitivo y el nombre del ganador absoluto.
